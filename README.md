# desk_tidy (v1.0 可用)

桌面整理工具，聚合桌面快捷方式/文件/文件夹，支持双击打开、右键移动/删除、隐藏项切换、主题与背景设置等。

## 当前完成度
- ✅ 应用/快捷方式：自动扫描桌面与公共桌面，提取图标，双击启动，右键“打开 / 使用其他应用打开 / 移动 / 删除到回收站”。`.lnk` 名称展示时去掉后缀。
- ✅ 全部视图：汇总桌面所有项，支持双击/右键操作。
- ✅ 文件夹视图：桌面根层仅显示文件夹；进入子目录后显示该目录的文件与子文件夹，右键同上。
- ✅ 文件视图：仅文件，双击默认打开，右键支持用其他应用、移动、删除。
- ✅ 外观与主题：浅色/深色/跟随系统，窗口透明度、图标大小、网格列数、隐藏项开关、自动刷新开关（占位）。
- ✅ 背景图片：设置页直接选择本地图片作为背景，可清除还原。
- ✅ Windows 特性：真实文件/快捷方式图标、默认程序打开、指定程序打开、删除到回收站。

## 使用
1. 运行 `flutter pub get`
2. 桌面端运行：`flutter run -d windows`
3. 交互：双击打开，右键操作；设置页选择背景图片等。

# desk_tidy

A Desktop organizing project.

## Getting Started
1. 显示模式分为两种  
   1. 所有的应用、文件、文件夹都显示在一个窗口里，通过选择 tab 切换  
   2. 风格化模式（当前未想到好的名字，点击任意屏幕可弹出右键菜单）  
      1. 默认分为三个屏，鼠标划到顶部时显示主屏，主屏内容为 app  
      2. 鼠标移动到屏幕最左侧一定范围时，显示文件  
      3. 鼠标滑动到屏幕最右侧时，显示文件夹  
      4. 待定功能：可以扩展分屏数量，例如顶部左侧是 app 里的编程软件，顶部中间是媒体编辑，顶部右侧是游戏。用户可以调整每个屏里的具体内容  
2. 用户可以将“全部”里的拖动到其他分屏  
3. 可以在其他分屏右键直接添加“全部”里的或者其他地方的文件，仅逻辑建立索引，并不是快捷方式或者硬链接  
4. 个性化  
   1. 提供主题修改  
   2. 可以设置背景颜色  

## 文件夹
1. 当前文件夹是在应用里实现了文件夹的操作逻辑，实际上可以点击文件夹的时候打开 Windows 的文件资源管理器
